# 요리조리 (YoriZori) 데모 시나리오

> **주제**: "냉장고 속 재료로 오늘 저녁 해결하기"
> **대상**: 자취생 / 1인 가구 사용자

---

## 전체 흐름

```
[1] 회원가입 & 온보딩
      ↓
[2] 냉장고에 재료 등록
      ↓
[3] 홈에서 날씨 확인 & 레시피 추천 받기
      ↓
[4] 레시피 검색으로 원하는 요리 찾기
      ↓
[5] 레시피 상세 보기 & 조리 타이머 사용
      ↓
[6] 영양 섭취 기록 & OCR로 가공식품 추가
      ↓
[7] 부족한 재료 장보기 목록으로 자동 생성
      ↓
[8] 마이페이지에서 즐겨찾기 확인
```

---

## 시나리오 1 — 회원가입 & 온보딩

**화면**: `AuthScreen` → `OnboardingScreen`

### 진행 순서

1. 앱 실행 → **회원가입** 탭 선택
2. 이메일 / 비밀번호 입력 후 가입
3. 온보딩 화면에서 신체정보 입력
   - 성별, 나이, 키, 몸무게
   - 목표 선택: `다이어트 / 유지 / 벌크업`
   - 활동량 선택: `낮음 / 보통 / 높음`
4. 완료 → 홈 화면 진입

### 핵심 기능 포인트

- PBKDF2 해시 기반 비밀번호 암호화 저장
- 신체정보는 `user_profiles` 테이블에 별도 저장
- 목표/활동량에 따라 일일 권장 영양 섭취량(DRI) 자동 계산

## 시나리오 2 — 냉장고 재료 등록

**화면**: `FridgeScreen`

### 진행 순서

1. 하단 탭 **냉장고** 선택
2. 우하단 **+** 버튼 → 재료 추가 모달
3. 재료명 입력: `달걀`, `두부`, `대파`, `간장`
4. 카테고리 선택: `계란 / 두부·콩류 / 채소 / 양념·소스`
5. 유통기한 설정: `3일 / 1주 / 2주 / 1개월` 프리셋 또는 직접 입력
6. 등록 완료 → 목록에 이모지와 함께 표시

### 핵심 기능 포인트

| 기능 | 설명 |
|---|---|
| 카테고리별 이모지 | 재료명 자동 인식 → 🥚 🫘 🧅 🫙 등 표시 |
| 유통기한 D-day 뱃지 | `D-3`, `오늘까지`, `기한 지남` 색상 구분 표시 |
| 기피 재료 등록 | 알레르기·싫어하는 재료 등록 → 레시피 추천에서 자동 제외 |
| 카테고리 탭 필터 | 채소 / 과일 / 육류 / 해산물 / 유제품 / 계란 / 두부·콩류 / 양념·소스 / 냉동식품 |

---

## 시나리오 3 — 홈 화면: 날씨 & 레시피 추천

**화면**: `HomeScreen`

### 진행 순서

1. 홈 화면 진입 시 **현재 위치 날씨** 자동 표시
   - 현재 기온, 최저/최고 기온, 날씨 아이콘
2. **냉장고 재료 기반 추천 레시피** 목록 표시
   - 달걀 + 두부 + 대파 보유 → `두부달걀찜`, `달걀찜`, `두부조림` 등 추천
3. 원하는 레시피 카드 탭 → 상세 화면 이동

### 핵심 기능 포인트

| 기능 | 설명 |
|---|---|
| 날씨 연동 | 기상청 API 기반 현재 위치 날씨 |
| 재료 매칭 점수 | 보유 재료가 많이 겹칠수록 상위 노출 |
| 부분 매칭 | 재료 일부만 있어도 추천 (완벽 매칭 > 부분 매칭) |
| Fallback | 재료 없어도 최신/랜덤 레시피 추천 |

## 시나리오 4 — 레시피 검색

**화면**: `RecipesScreen`

### 진행 순서

1. 하단 탭 **레시피** 선택
2. 검색창에 `두부` 입력
3. 결과 목록: `두부조림`, `순두부찌개`, `두부달걀찜` 등 표시
4. 칩 필터로 좁히기: `국·찌개 / 볶음 / 찜 / 구이`
5. 원하는 레시피 선택

### 핵심 기능 포인트

- 식약처 COOKRCP01 기반 **1,000개 이상** 레시피 DB
- 재료명 동의어 처리: `계란` ↔ `달걀`, `대파` → `파` 자동 통합
- 공백 제거 후 비교 → 오타·띄어쓰기 차이 허용

## 시나리오 5 — 레시피 상세 & 조리 타이머

**화면**: `RecipeDetailScreen`

### 진행 순서

1. 레시피 선택 → 상세 화면 진입
2. **영양 정보** 확인: 칼로리 / 탄수화물 / 단백질 / 지방 / 나트륨
3. **재료 목록** 확인: 보유 재료는 초록 표시, 없는 재료는 회색 표시
4. **조리 단계** 스크롤: 단계별 설명 + 이미지
5. **타이머 버튼** 탭
   - 분/초 직접 입력 또는 프리셋 선택
   - 시작 → MM:SS 카운트다운 표시
   - 완료 시 **벨 소리 + 진동** 알람
6. **즐겨찾기** 버튼으로 저장
7. **식사 기록 추가** → 식사 유형 선택 (아침/점심/저녁/간식) → 인분 수 입력 → 기록

### 핵심 기능 포인트

| 기능 | 설명 |
|---|---|
| 재료 보유 표시 | 냉장고와 실시간 대조 → 있는 재료 / 없는 재료 색상 구분 |
| 조리 타이머 | MM:SS 표시, 테두리 스냅 애니메이션, 색상 선택 |
| 알람 | `expo-av` 벨 소리 + `expo-haptics` 진동 |
| 즐겨찾기 | `favorites` 테이블에 저장 |
| 인분 계산 | 입력한 인분에 비례해 영양정보 자동 계산 |

## 시나리오 6 — 영양 기록 & OCR

**화면**: `NutritionScreen`

### 진행 순서

**[레시피 기반 기록]**

1. 하단 탭 **영양** 선택
2. 오늘 날짜 영양 요약 카드 확인
   - 칼로리 / 탄수화물 / 단백질 / 지방 / 나트륨 게이지 바
3. 레시피 상세에서 기록한 `두부달걀찜 1인분` 반영 확인
4. ◀ ▶ 버튼으로 날짜 이동

**[OCR 가공식품 기록]**

1. **+ 음식 추가** → **사진 촬영** 선택
2. 편의점 삼각김밥 뒷면 영양성분표 촬영
3. Google Vision API → 칼로리·탄수화물·단백질·지방·나트륨 자동 인식
4. 인식 결과 확인 후 **저장** → `custom_foods` 및 `frequent_products`에 등록
5. 다음번부터 **자주 먹는 음식** 목록에서 바로 선택 가능

### 핵심 기능 포인트

| 기능 | 설명 |
|---|---|
| DRI 기반 목표 | 온보딩 신체정보 기반 일일 권장 섭취량 표시 |
| 아침/점심/저녁/간식 구분 | 시간대별 섭취 내역 분리 표시 |
| OCR 자동 인식 | Google Vision API로 영양성분표 숫자 추출 |
| 자주 먹는 음식 | OCR 결과 저장 → 재선택 편의 제공 |
| 날짜 이동 | ◀ ▶ 로 어제/내일 기록 조회 |

## 시나리오 7 — 장보기 목록

**화면**: `ShoppingScreen`

### 진행 순서

1. 하단 탭 **장보기** 선택
2. **레시피에서 자동 추가된 항목** 확인
   - 레시피 상세에서 없는 재료로 표시된 항목이 자동 추가됨
   - 예: `순두부`, `멸치액젓`
3. **직접 추가**: `두유` 수동 입력
4. 각 항목 이모지 자동 표시 (재료명 기반)
5. 마트에서 구매 후 체크박스 탭 → 취소선 처리
6. **쿠팡 바로가기** 버튼 탭 → 해당 재료 쿠팡 검색 페이지 열기
7. 전체 구매 완료 → **완료된 항목 일괄 삭제**

### 핵심 기능 포인트

| 기능 | 설명 |
|---|---|
| 자동 생성 | 레시피 재료 중 냉장고에 없는 것만 자동 추가 |
| 기피 재료 제외 | 등록된 기피 재료는 장보기 목록에서 자동 제외 |
| 쿠팡 연동 | 항목명으로 쿠팡 검색 URL 자동 생성 후 이동 |
| 이모지 표시 | 재료명 자동 인식 → 카테고리별 이모지 표시 |

## 시나리오 8 — 마이페이지

**화면**: `MyPageScreen`

### 진행 순서

1. 하단 탭 **마이페이지** 선택
2. 프로필 정보 확인 (닉네임, 이메일)
3. **즐겨찾기 레시피** 목록 확인 → 바로 상세 이동 가능
4. 신체정보 수정 (키/몸무게 변경 → DRI 재계산)
5. 로그아웃

## 기능 요약표

| 화면 | 핵심 기능 | 사용 데이터 |
|---|---|---|
| 회원가입/로그인 | JWT 인증, 소셜 로그인 | `users` |
| 온보딩 | 신체정보 입력, DRI 계산 | `user_profiles` |
| 홈 | 날씨 표시, 냉장고 기반 추천 | `recipes`, `pantry_items` |
| 레시피 검색 | 키워드·재료명 검색, 카테고리 필터 | `recipes`, `recipe_ingredients` |
| 레시피 상세 | 재료 보유 표시, 타이머, 즐겨찾기 | `recipe_steps`, `favorites` |
| 냉장고 | 재료 등록, D-day 뱃지, 기피 재료 | `pantry_items`, `avoid_ingredients` |
| 영양 기록 | 식사 기록, OCR, 자주 먹는 음식 | `nutrition_logs`, `custom_foods`, `frequent_products` |
| 장보기 | 자동 생성, 체크, 쿠팡 연동 | `shopping_items` |
| 마이페이지 | 즐겨찾기, 프로필 수정, 로그아웃 | `users`, `favorites` |
