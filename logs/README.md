# 📋 로그 파일 폴더

요리조리 프로젝트의 모든 로그 및 변경 기록을 관리하는 폴더입니다.

## 📁 파일 설명

| 파일                  | 설명                                        |
| --------------------- | ------------------------------------------- |
| **ErrorLog.md**       | UI 스모크 테스트 결과 및 발견된 에러 로그   |
| **ERROR_LOG.md**      | 에러 로그 및 스모크 테스트 상세 분석 (한글) |
| **CHANGELOG_JoRi.md** | JoRi 브랜치의 작업 내역 및 변경 사항        |

## 📝 파일 사용법

### ErrorLog.md

- localhost:8082 UI 스모크 테스트 결과
- 발견된 에러 (E-001, E-002, E-003)
- 환경 설정 및 재실행 명령어

### ERROR_LOG.md

- 한글 기반 에러 로그
- 스모크 테스트 결과 요약
- 후속 테스트 계획 포함

### CHANGELOG_JoRi.md

- 작업일 기준 변경 사항 기록
- 백엔드/프론트엔드 수정 내역
- 새로 추가된 기능 설명

## 🔄 유지보수

새로운 오류가 발생하거나 주요 변경 사항이 있을 때 이 폴더의 파일을 업데이트하세요.
